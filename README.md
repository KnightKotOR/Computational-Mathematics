**Численные методы и вычислительная математика** — реализация алгоритмов решения математических задач

Сборник лабораторных работ и курсового проекта, посвященных численному решению уравнений, систем дифференциальных уравнений, обработке матриц и моделированию физических процессов.

## О проекте

Проект включает в себя выполненные лабораторные работы (вариант 31) и курсовую работу по дисциплине «Вычислительная математика» (СПбПУ им. Петра Великого). Решения реализованы на языке Python с использованием научных библиотек SciPy и NumPy.

### Ключевые возможности

- **Лабораторная №1:** Построение кубических сплайнов (`CubicSpline`) для таблично заданных функций. Нахождение корней нелинейных уравнений методами бисекции (`bisect`) и Брента (`brentq`).
- **Лабораторная №2:** LU-разложение матриц, вычисление обратных матриц и их норм. Исследование зависимости числа обусловленности матрицы (`cond`) от ее параметров.
- **Лабораторная №3:** Решение систем обыкновенных дифференциальных уравнений (ОДУ) методами Рунге-Кутты 4-го порядка (RK4) и Рунге-Кутты-Фельберга (RKF45 / `dopri5`).
- **Курсовая работа:** Моделирование электрической цепи. Доказательство соответствия системы ОДУ законам Кирхгофа, нахождение параметров цепи и оценка емкости конденсатора $C$ путем минимизации среднеквадратичного отклонения.

### Результаты

- Экспериментально доказано, что метод Брента обладает большей скоростью сходимости по сравнению с методом бисекции.
- При уменьшении параметра $p$ (Лаб. №2) показан рост числа обусловленности матрицы, что приводит к вычислительному коллапсу при нахождении обратной матрицы.
- В курсовой работе получена оценка емкости конденсатора $C = 1$ мкФ с среднеквадратичным отклонением $\approx 0.00028$. Подтверждено, что погрешность округления исходных данных не влияет на итоговый результат вычисления $C$.

## Установка и запуск

### Предварительные требования
- [Python](https://www.python.org/downloads/) версии 3.10 или выше

### 1. Клонирование репозитория

```commandline
git clone https://github.com/KnightKotOR/Computational-Mathematics.git
```

### 2. Установка зависимостей

Создайте виртуальное окружение и установите необходимые библиотеки:

```commandline
python -m venv venv
venv\Scripts\activate
```

```commandline
pip install numpy scipy matplotlib prettytable
```

### 3. Запуск скриптов

Перейдите в папку с нужной работой и запустите соответствующий Python-файл:

```commandline
python lab1/main.py
```
